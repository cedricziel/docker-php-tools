# docker-php-tools

Base Image for testing PHP applications with additional tools installed.

## Pre-installed PHP extensions

* xdebug
* xsl
* zip

## Installed Tools

### Build

* ant

### Dependency Management

* composer

### Documentation

* pandoc
* texlive

### Packages

* libxslt-dev
* Google Cloud SDK

### Version control

* git
